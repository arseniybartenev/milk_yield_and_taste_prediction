## Описание проекта

Фермер хозяйства «Вольный луг» планирует расширить поголовье стада, приобретая коров у ассоциации «ЭкоФерма». Условия контракта позволяют тщательно отбирать животных: каждая бурёнка должна давать не менее 6000 кг молока в год, а надой должен быть вкусным по строгим критериям фермера. «ЭкоФерма» предоставила подробные данные о своих коровах.

Цель проекта — разработать две модели машинного обучения, которые помогут фермеру управлять рисками и принимать обоснованные решения о покупке:

- **Модель регрессии** для прогнозирования годового удоя (целевой признак `Удой`).
- **Модель классификации** для определения качества вкуса молока (целевой признак `Вкус молока`, бинарный – вкусно / не вкусно).

## Данные

Предоставлено три CSV-файла:

- `ferma_main.csv` – данные о текущем стаде фермера (характеристики коров, кормления, молока).
- `ferma_dad.csv` – информация об отцах коров (связь по `id`).
- `cow_buy.csv` – данные о коровах «ЭкоФермы», которые рассматриваются к покупке (аналогичные признаки, но без целевых переменных).

Данные требуют предобработки: изменение форматов, обработка пропусков (отсутствуют), удаление явных и неявных дубликатов, кодирование категориальных признаков, масштабирование.

## Этапы работы

1. **Загрузка и первичный анализ** – изучение структуры данных, выявление ошибок.
2. **Предобработка** – приведение типов, удаление дубликатов, унификация категориальных значений, кодирование, обработка выбросов.
3. **Исследовательский анализ (EDA)** – визуализация распределений, корреляционный анализ, отбор признаков.
4. **Построение моделей регрессии** для прогноза удоя (LinearRegression, метрики RMSE, MAE, R²).
5. **Построение моделей классификации** для предсказания вкуса молока (LogisticRegression, метрики accuracy, precision, recall, F1).
6. **Оценка и выбор лучших моделей** на тестовых данных.
7. **Формирование рекомендаций** по отбору коров.

## Используемые технологии

## Результаты

### Регрессия (удой)

- Лучшая модель – линейная регрессия с признаками `feed_energy`, `feed_energy²`, `sugar_protein_ratio` (категориальный), `crude_protein`, `dads_name`.
- Модель позволила спрогнозировать удой с достаточной точностью для отбора коров с ожидаемым надоем >6000 кг.

### Классификация (вкус молока)

- Лучшая модель – логистическая регрессия с порогом принятия решения **0.8**.
- При таком пороге модель даёт только один ложноположительный прогноз, что минимизирует риск покупки коровы с невкусным молоком.

### Итоговые рекомендации

На пересечении предсказаний обеих моделей выделены **две коровы**, которые с высокой вероятностью обеспечат и нужный удой, и отличный вкус молока. Их можно рекомендовать к покупке с минимальным риском для фермера.

## Выводы

Разработанные модели позволяют фермеру:

- количественно оценивать потенциальный удой каждой коровы;
- определять, будет ли её молоко соответствовать вкусовым критериям;
- отбирать животных с наилучшим сочетанием показателей, снижая финансовые риски и повышая эффективность хозяйства.

Проект подтвердил, что качественная предобработка данных и разумное конструирование признаков (квадрат ЭКЕ, категоризация СПО, использование информации об отцах) способны значительно улучшить качество моделей даже при использовании простых алгоритмов (линейная регрессия, логистическая регрессия).

## Запуск проекта

1. Клонируйте репозиторий:
   ```bash

   ```bash
   cd milk_yield_and_taste_prediction
   ```

   ```bash

   ```

   ```
